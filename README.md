## ConsoleAppStandard
Standard Fizz Buzz for range `1` to `100`.

## ConsoleAppTwisted
Uses these tokens:
```
{
    new Token( 5, "Fizz"),
    new Token( 9, "Buzz"),
    new Token(27, "Bar")
}
```
for range `-20` to `127`.

## TwistedFizzBuzz
This is the library.
